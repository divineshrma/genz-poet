# 🚀 Gen-Z Slang Sentiment Analyzer

## 📖 About The Project
A lightweight Python tool that analyzes the emotional tone (sentiment) of modern Gen-Z digital communication and slang. 

This technical project is a practical extension of my academic research paper, **"Gen-Z Themes in Modern Poetry, Fiction, and Drama: A Digital-First Literary Revolution"**. While my paper explored linguistic shifts qualitatively, this tool uses Natural Language Processing (NLP) to decode those communication patterns programmatically.

## 🛠️ Tech Stack
* **Language:** Python
* **Library:** TextBlob (for NLP and Sentiment Analysis)

## 💡 How It Works
The script takes any Gen-Z slang or modern sentence as input and calculates its polarity score to determine if the "vibe" is:
* **Positive ✨** (e.g., "This outfit is lowkey fire, W")
* **Negative 🚩** (e.g., "That behavior is so cringe, L")
* **Neutral 😐** (e.g., "It was totally mid, NPC energy")

## 🚀 How to Run
1. Clone this repository.
2. Install the required library: `pip install textblob`
3. Run the script: `python main.py`
4. Enter your text and check the vibe!
